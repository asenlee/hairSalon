# HairSalon

## To start Server:
>cd hairSalon-node
1. >npm i --save (this installs all the dependencies)
2. >npm run start


## To start Client:
>cd hairSalon-angular
1. >npm i --save
2. >npm run start
3. go to http://localhost:4200/
